EXPENSE TRACKER

A full-stack Expense Tracking Application that allows users to add expenses, view monthly summary, and analyze category-wise spending using charts.

FEATURES
Add Expense

Enter Title, Amount, Date, Category, and Notes
Stored securely in a SQLite database

Monthly Summary

Displays all expenses of the current month
Shows Title, Amount, Date, Category, Notes
Auto-calculates Total Monthly Spending

Expense Analysis

Visual chart showing expenses grouped by category
Helps understand spending patterns
